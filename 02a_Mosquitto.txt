    i) --------------------------------------------------------------------DigitalOcean

-> Se você ainda não possui uma conta, crie utilizando este código:

	https://m.do.co/c/c3ec30ab3a61
	

-> Crie um "droplet" Ubuntu com 1 GB de RAM e 30 GB de disco (Custo: $10/mês ou $0,015/hora)

	-> Dê-lhe um "alias" de fácil rememoração
	-> Uma vez criado, anote o IP público e faça uma conexão via ssh usando o Putty
	-> Você vai receber um email contendo o endereço IP público, bem como a senha do usuário root
	-> Assim que a conexão acontecer, uma nova senha para o usuário root será pedida


   ii) --------------------------------------------------------------------Criação de usuário

groupadd -g 1400 mosquittogrp
mkdir -p /home/mosquittousr
useradd -u 1500  -g mosquittogrp -d /home/mosquittousr mosquittousr
passwd mosquittousr 
<< senha >>

chown -R mosquittousr:mosquittogrp /home/mosquittousr
sudo adduser mosquittousr sudo

  iii) --------------------------------------------------------------------Instalação Mosquitto
su - mosquittousr
sudo apt-get update
sudo apt-get install mosquitto mosquitto-clients

-> arquivo de configuração gerado em /etc/mosquitto/mosquitto.conf

   iv) --------------------------------------------------------------------Assinatura

-> Em outra sessão putty, conectado como mosquittousr:

mosquitto_sub -h localhost -t pubteste

    v) --------------------------------------------------------------------Publicação

mosquitto_pub -h localhost -t pubteste -m "Texto exemplo!"

   vi) --------------------------------------------------------------------Acompanhamento


ps -efH | grep mosquitto.conf | grep -v grep | wc -l

  vii) --------------------------------------------------------------------Interrupção 

sudo systemctl stop mosquitto
-- também poderíamos ter utilizado o comando kill -9 <<número processo serviço mosquitto>>

 viii) --------------------------------------------------------------------Reinicio 

sudo systemctl restart mosquitto
